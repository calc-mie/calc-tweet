Usage :: calc-tweet user
 calc-tweet user add <screen_name> @<bot`s screen_name> ... add user
 calc-tweet user rm <screen_name> @<bot`s screen_name> ... remove user
 calc-tweet user show <screen_name> @<bot`s screen_name> ... show groups which <screen_name> join
 calc-tweet user help @<bot`s screen_name> ... show this manual on direct message
